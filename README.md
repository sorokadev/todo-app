# Full-Stack Todo List Application

A complete full-stack todo list application built with React (Next.js) on the frontend and NestJS on the backend, both using TypeScript and GraphQL.

## Features

- Create new todo items
- View a list of all todo items
- View detailed information about a specific todo item
- Mark todo items as completed/uncompleted
- Delete todo items
- Fully typed with TypeScript
- GraphQL API with Apollo Client/Server

## Technologies Used

### Frontend

- Next.js (React framework)
- Apollo Client (GraphQL client)
- TypeScript
- Tailwind CSS
- shadcn/ui components

### Backend

- NestJS (Node.js framework)
- Apollo Server (GraphQL server)
- TypeScript

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (v16 or later)
- npm (v7 or later)

## Installation

cd backend
npm install --legacy-peer-deps
npm run start:dev

cd ../
npm install --legacy-peer-deps
npm run dev
